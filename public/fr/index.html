Thème Hugo learn

Hugo-theme-learn est un thème pour Hugo, un générateur de site statique, rapide et modern, écrit en Go. Tandis que Hugo est souvent utilisé pour des blogs, ce thème multi-langue est entièrement conçu pour la documentation.

Ce thème est un portage partiel du thème Learn de Grav, un CMS modern écrit en PHP.

Le thème Learn fonctionne grâce à la structure de page aborescentes pour organiser le contenu: tous les contenus sont des pages qui appartiennent à d’autres pages. Plus d’infos

Fonctionnalités principales

  • Recherche automatique
  • Mode multi-langue
  • Nombre de niveau infini dans le menu
  • Boutons suivant/précédent automatiquement générés pour naviguer entre les items du menu
  • Taille d’image, ombres…
  • Fichiers joints
  • Lister les pages filles
  • Diagrammes Mermaid (flowchart, sequence, gantt)
  • Style configurable and variantes de couleurs
  • Boutons, Messages Astuce/Note/Info/Attention, Expand

Contribuer à cette documentation

Site de documentation

Cette documentation statique a été générée avec Hugo avec une simple commande : hugo -t hugo-theme-learn – le code source est disponible sur Github

Le site est auomatiquement publié et hébergé par Netlify. Plus d’infos sur le déploiement de site Hugo avec Netlify(En anglais)
